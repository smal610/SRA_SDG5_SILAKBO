# SILAKBO: Women-Only Harassment Reporting and Support System

## Project Title and SDG Goal
SILAKBO is a C# Windows Forms application that provides a safe, secure, and confidential platform for women to report harassment incidents and access support resources.

SDG Alignment: United Nations Sustainable Development Goal 5 – Achieve gender equality and empower all women and girls.  
Target 5.2: Eliminate all forms of violence against women and girls in public and private spheres.

## Project Description
SILAKBO allows users to:
- Register and login securely using hashed passwords
- Submit harassment reports with optional evidence
- Track the status of their reports
- View support resources for assistance

Administrators can:
- View all submitted reports
- Update report statuses
- Backup reports using JSON

The system follows an N-Tier Architecture (Presentation, Business Logic, Data Access).

## Technologies Used
- C# (.NET Windows Forms)
- MySQL Database
- ADO.NET (MySql.Data)
- JSON Serialization
- Async/Await for performance

## Installation
1. Clone the repository
2. Open the solution in Visual Studio
3. Configure database connection in DBConnection.cs
4. Build and run the project

## Database Tables
Users, Reports, SupportResources
